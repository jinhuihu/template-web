### 3️⃣ 编写代码（3分钟）

```css
/* 设计稿标注多少px，代码就除以100写成rem */

/* 例子：设计稿标注宽度690px，高度200px，内边距30px */
.card {
  width: 6.9rem;      /* 690 ÷ 100 = 6.9 */
  height: 2rem;       /* 200 ÷ 100 = 2 */
  padding: 0.3rem;    /* 30 ÷ 100 = 0.3 */
  font-size: 14px;    /* ⚠️ 字体用px，不用rem */
  border: 1px solid #ddd; /* ⚠️ 边框用px，不用rem */
}
```

**就这么简单！** ✅

## 🎯 三条黄金规则

### 规则1：计算rem

```
rem = 设计稿px ÷ 100
```

### 规则2：字体用px

```css
font-size: 14px;  /* ✅ */
font-size: 0.14rem; /* ❌ */
```

### 规则3：边框用px

```css
border: 1px solid #ddd;  /* ✅ */
border: 0.01rem solid #ddd; /* ❌ */
```

## 📐 常用尺寸（直接用）

```css
/* 宽度 */
width: 7.5rem;    /* 750px 全屏 */
width: 6.9rem;    /* 690px 内容区（推荐）*/
width: 3.45rem;   /* 345px 半屏 */

/* 高度 */
height: 0.88rem;  /* 88px 导航/输入框/按钮 */
height: 1rem;     /* 100px 大按钮 */
height: 1.2rem;   /* 120px 列表项 */

/* 间距 */
padding: 0.3rem;  /* 30px 常用间距 */
padding: 0.2rem;  /* 20px 小间距 */
margin: 0.3rem;   /* 30px 外边距 */

/* 圆角 */
border-radius: 0.16rem; /* 16px 卡片圆角 */
border-radius: 0.08rem; /* 8px 按钮圆角 */
border-radius: 50%;     /* 圆形 */

/* 字体（用px）*/
font-size: 18px;  /* 大标题 */
font-size: 16px;  /* 标题 */
font-size: 14px;  /* 正文 */
font-size: 12px;  /* 小字 */
```
