💡 使用说明

开发模式（npm run dev）：

• 图片直接从远程加载

• URL不会被替换

• 方便快速开发


生产构建（npm run build）：

• 自动下载远程图片

• 保存到 dist/assets/images/[页面名]/

• 替换URL为本地路径

• 生成稳定的部署包

🧪 验证步骤

1. 运行生产构建：npm run build

2. 检查图片目录：ls dist/assets/images/image-test/

3. 查看HTML源码：cat dist/image-test.html | grep "img src"

4. 打开页面验证图片显示正常
